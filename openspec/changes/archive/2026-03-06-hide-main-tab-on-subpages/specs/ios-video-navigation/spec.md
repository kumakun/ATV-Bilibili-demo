## 新增需求

### 需求：进入视频详情页时必须隐藏主 Tab Bar
系统必须在用户进入视频详情页时隐藏主 Tab Bar，并在用户离开视频详情页返回上一级后恢复显示主 Tab Bar。

#### 场景：从关注页进入视频详情页
- **当** 用户在关注页点击视频卡片并进入视频详情页
- **那么** 系统导航到对应的视频详情页
- **那么** 系统隐藏主 Tab Bar

#### 场景：从收藏页进入视频详情页
- **当** 用户从收藏夹列表进入任意视频详情页
- **那么** 系统导航到对应的视频详情页
- **那么** 系统隐藏主 Tab Bar

#### 场景：从视频详情页返回上一级
- **当** 用户从视频详情页返回原列表页
- **那么** 系统恢复显示主 Tab Bar
- **那么** 系统保持原列表页的内容状态和滚动位置
