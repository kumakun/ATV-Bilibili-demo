## 新增需求

### 需求：主 Tab Bar 必须仅在一级页面显示
主 Tab Bar 必须仅在“关注 / 收藏 / 我的”三个一级根页面显示。当用户在任一主 Tab 内通过 push 导航进入非根页面时，系统必须隐藏主 Tab Bar，避免一级导航干扰二级页面内容。

#### 场景：从一级页面进入二级页面时隐藏主 Tab Bar
- **当** 用户在“关注 / 收藏 / 我的”任一一级页面中点击可 push 的子页面入口
- **那么** 系统进入目标二级页面
- **那么** 系统隐藏主 Tab Bar

#### 场景：从二级页面返回一级页面时恢复主 Tab Bar
- **当** 用户从二级页面返回所属主 Tab 的一级根页面
- **那么** 系统恢复显示主 Tab Bar
