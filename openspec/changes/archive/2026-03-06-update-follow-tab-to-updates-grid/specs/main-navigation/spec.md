## 修改需求

### 需求：主 Tab Bar 必须包含 3 个导航标签

应用的主导航栏必须包含且仅包含以下 3 个标签页，按照指定顺序排列：更新、收藏、我的。系统禁止在主导航中显示"推荐"、"热门"、"排行榜"标签，也禁止将第一项继续显示为“关注”。

#### 场景：应用启动时显示正确的 Tab 数量

- **当** 用户启动应用并成功登录
- **那么** 系统必须在主 Tab Bar 中显示恰好 3 个标签页

#### 场景：Tab 顺序正确

- **当** 用户查看主 Tab Bar
- **那么** 系统必须按以下顺序从左到右显示标签：
  1. 更新
  2. 收藏
  3. 我的

#### 场景：推荐热门排行榜标签不可见

- **当** 用户查看主 Tab Bar 中的所有标签
- **那么** 系统禁止显示"推荐"、"热门"、"排行榜"标签

#### 场景：第一项标签文案已更新

- **当** 用户查看主 Tab Bar 的第一项入口
- **那么** 系统必须显示“更新”文案
- **那么** 系统禁止显示旧文案“关注”作为该主入口名称
