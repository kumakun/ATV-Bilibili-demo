## 新增需求

### 需求：点赞功能
系统必须支持用户对视频进行点赞和取消点赞操作。

#### 场景：点赞视频
- **当** 用户点击点赞按钮且当前未点赞
- **那么** 系统发送点赞请求并更新UI状态为已点赞

#### 场景：取消点赞
- **当** 用户点击点赞按钮且当前已点赞
- **那么** 系统发送取消点赞请求并更新UI状态为未点赞

#### 场景：显示点赞状态
- **当** 视频信息加载完成
- **那么** 系统显示当前用户是否已点赞该视频

### 需求：投币功能
系统必须支持用户为视频投币，可选择投1个或2个硬币。

#### 场景：投币操作
- **当** 用户点击投币按钮并选择数量
- **那么** 系统发送投币请求并更新UI状态

#### 场景：投币数量限制
- **当** 用户已为该视频投过币
- **那么** 系统禁止重复投币并显示已投币状态

#### 场景：投币失败处理
- **当** 投币请求失败（如硬币不足）
- **那么** 系统显示错误提示

### 需求：收藏功能
系统必须支持用户收藏视频到收藏夹。

#### 场景：收藏视频
- **当** 用户点击收藏按钮
- **那么** 系统显示收藏夹列表供用户选择

#### 场景：取消收藏
- **当** 用户取消对视频的收藏
- **那么** 系统更新收藏状态

#### 场景：显示收藏状态
- **当** 视频信息加载完成
- **那么** 系统显示该视频是否已被收藏

### 需求：关注UP主
系统必须支持用户关注和取关UP主。

#### 场景：关注UP主
- **当** 用户点击关注按钮且当前未关注
- **那么** 系统发送关注请求并更新UI为已关注状态

#### 场景：取消关注
- **当** 用户点击关注按钮且当前已关注
- **那么** 系统发送取消关注请求并更新UI为未关注状态

#### 场景：显示关注状态
- **当** 视频信息加载完成
- **那么** 系统显示当前用户是否已关注该UP主

### 需求：分享功能
系统必须支持用户分享视频链接。

#### 场景：打开分享面板
- **当** 用户点击分享按钮
- **那么** 系统显示iOS原生分享面板

#### 场景：分享视频链接
- **当** 用户选择分享方式
- **那么** 系统生成视频链接并通过所选方式分享

### 需求：互动操作权限
系统必须验证用户登录状态才能进行互动操作。

#### 场景：未登录时操作
- **当** 未登录用户尝试点赞、投币、收藏或关注
- **那么** 系统提示用户需要先登录

#### 场景：已登录时操作
- **当** 已登录用户进行互动操作
- **那么** 系统正常执行操作
