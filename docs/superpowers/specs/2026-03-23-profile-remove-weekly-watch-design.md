# 我的页移除每周必看设计

**日期：** 2026-03-23

## 目标

从 iOS 项目的“我的”页面移除“每周必看”功能，并清理只服务于该功能的页面、状态、模型与接口桥接代码。

## 范围

- 删除 iPhone 与 iPad 我的页中的“每周必看”入口。
- 删除 `ProfileRoute` 中的 `weeklyWatch` 路由。
- 删除 iOS 侧 `WeeklyWatchView` 与 `WeeklyWatchViewModel`。
- 删除 iOS 侧“每周必看”专用模型和接口桥接方法。
- 保持 tvOS 项目不变。

## 设计

当前“每周必看”在 iOS 内是一条独立链路，没有被其他模块复用。删除时顺带把 Profile 功能入口的重复配置收敛为统一的主功能路由列表与菜单元数据，避免 iPad 侧栏与 iPhone 功能列表继续各自维护一份配置。

## 验证

- 增加一个单元测试，约束主功能路由列表不包含“每周必看”。
- 使用 Xcode MCP 运行相关单元测试。
- 使用 Xcode MCP 构建 `BilibiliLive-iOS`，确认删除后工程仍可编译。
